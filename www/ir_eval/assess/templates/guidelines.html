Before making the judgement, please read the following instructions carefully:

  1. There are TWO categories of review sentences:
    • : Bug Report
    • : Not Bug Report
  2. A sentence should be judged as Bug Report review if:
    • it clearly states the functional limitation of the app;
    • or, it describes a situation when the app fails to work properly.
  3. A sentence should be judged as Not Bug Report review if:
    • it can not be categorized into the previous group;
    • or, it is not written in English. Please note, if the sentence is not written in English, even if you could understand it, it should be judged as Not Bug Report.
  4. Be specific. If the sentence is a bug report, please choose the aspect that the sentence is commenting about from the dropdown list. If none of the provided aspects is appropriate, please choose "Others" and specify up to 5 key words about what function this sentence is commenting.
  5. Be consistent. If one of the sentences is judged as Bug Report, the entire reviews should be considered as Bug Report as well. Use the same rule when judge different queries.
  6. Know the App first. The description is provided for each app on the review list page. Please read it first to get familiar with the App.
  7. Be strict. If you think an app falls in the border of two categories, choose Not Bug Report
